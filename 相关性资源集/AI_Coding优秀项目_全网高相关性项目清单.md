# AI Coding 优秀项目（本仓库）— 全网高相关性项目集合

> **说明**：本文档以「AI coding 成品项目观摩」为锚点，汇总别人用 Claude Code、Cursor、Lovable、Bolt、Replit Agent、Codex、Copilot Agent 等 AI coding / vibe coding 工作流做出的优秀项目。  
> **排序规则**：按 **与本项目主题的相关性得分（10 分制，高→低）** 降序排列；同分优先「可体验、有明确 AI coding 证据、形态完整、学习价值高」的项目。

**相关性打分维度（综合）**
1. 是否明确属于 **AI coding / vibe coding / agentic coding** 产物；
2. 是否是 **成品项目**，而不是工具、教程或资料入口；
3. 是否 **可直接体验或有源码可读**；
4. 是否覆盖有学习价值的产品形态：游戏、网站、App、扩展、个性化工具、特殊实验；
5. 是否适合拆解：交互完整度、技术实现、产品切口、增长/商业化可能性。

**证据等级**

| 等级 | 含义 |
|:---:|:---|
| A | 作者、README、应用商店或项目页明确说明 AI coding / vibe coding / Claude Code / Cursor 等参与开发 |
| B | 被可信同类清单收录，项目可体验，但原始作者证据仍建议复核 |
| C | 有社区传播或强线索，需要后续人工确认 |

## 使用建议

### 按学习目标选择项目

| 学习目标 | 推荐项目组合 |
|:---|:---|
| **学习 AI coding 做游戏** | Society Fail + Tower of Time + Vibe Sail + Oyster Arcade |
| **快速做一个有用的小产品** | How Many Layers + Standup Buddy + Lash Tracker + My Baby Logger |
| **学习生成式 AI Web App 包装** | MenuGen + JeniCards + Timeless Memories + Stories of Life |
| **学习真实业务闭环** | PrintPigeon + CourtSync Calendar + DummyForms |
| **学习 App/扩展上架形态** | Poker Slam + Crashy Zorg + Markdown Printer + TCKR |

### 拆解路径建议

1. **先看入口**：项目首页 30 秒内是否说清楚价值。
2. **再看输入**：它让用户输入文本、图片、文件、位置、日程，还是只点按钮。
3. **再看闭环**：结果是预览、下载、分享、购买、保存，还是只展示一次。
4. **再看状态**：登录、历史记录、错误处理、空状态、加载态是否完整。
5. **最后看源码**：有源码的优先拆数据模型、路由、组件边界、API 调用和部署方式。
